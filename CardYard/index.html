Card Yard an MTG Card Tracker app

Project Description

The Card Yard is an app made for the Project management course at NSCC - Designed to be able to detect cards with a phone attached camera and deposit them into a database allowing a player to keep track of their possibly large collection of Magic the Gathering cards.

Check it out on Github

Project Details

  • Android
  • Optical Character Recognition
  • Team based Development using Agile
